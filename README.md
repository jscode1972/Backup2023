## git 指令
```
$ git clone https://xxxx/xxxx.git
$ rm -rf .git

# 提交
$ git add .
$ git commit -m "說明..."
$ git push
$ git push -u master

# 反悔, 取消上述動作
$ git reset HEAD

# 如果衝突,先把上面的東西拉下來
$ git pull
跳出內容,編輯 :wq
$ git add . &&  git commit -m "說明..."
```
