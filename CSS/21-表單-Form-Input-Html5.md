## 表單輸入
```html
<input type="text" name="文字"> 
<input type="password" name="密碼"> 
<input type="hidden" name="隱藏"> 
<input type="submit" value="送出"> 
<input type="reset" value="清除"> 
<input type="button" name="按鈕"> 
<input type="file" name="檔案上傳"> 
<input type="image" src="btn.png" alt="圖片"> 
<textarea name="">文字輸入</textarea>

<input type="radio" name="單選" value="1" checked> aaa
<input type="radio" name="單選" value="1"> bbb

<input type="checkbox" name="多選" value="1" checked> aaa
<input type="checkbox" name="多選" value="2" > aaa
<input type="checkbox" name="多選" value="3" checked> aaa

<select name="複選" multiple>
  <option value="1" selected>選項一</option>
  <option value="2">選項二</option>
  <option value="3">選項三</option>
</select>
```
