
## 盒子
``` css
box {

  margin: 40px auto; /* auto -> 兩邊間距一樣 */
  
  border: 1px dotted #94c8b1;
  border-left: 複寫;
  
  border-shadow:         /* 陰影, 水平x,垂直y 模糊強度 擴散強度 陰影色 內側 */
  border-radius:         /* 圓角 */
 
 
  background-image: linear-gradient(to bottom, #1, #2);
 
}

:link      /* 未經點過 */
:visited   /* 曾經點過 */
:hover     /* 滑鼠移入 */
:active    /* 點擊滑鼠 */
:focus {   /* 取得焦點 */
}

文字 {
  font-family: 
  font-size: 
  font-weight: 
  font-style: 
  
  text-align: center;  /* 對齊 */
  text-decoration:     /* 底線 */
  text-indent:         /* 縮排 */
  text-shadow:         /* 陰影 */
  
  line-height:         /* 行高 */
  
}
```
